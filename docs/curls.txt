Create a new pipeline:
curl -X POST http://localhost:3000/api/pipeline -H "Content-Type: application/json" -d '{"firm_name": "American Discovery Capital"}'

Get single pipeline (replace {id} with actual pipeline ID):
curl -X GET http://localhost:3000/api/pipeline/{id}

Get all pipelines:
curl -X GET http://localhost:3000/api/pipeline

Get pipelines with filters:
# Filter by status
curl -X GET "http://localhost:3000/api/pipeline?status=pending"

# Filter by firm name
curl -X GET "http://localhost:3000/api/pipeline?firm_name=Test%20Firm"

# Multiple filters
curl -X GET "http://localhost:3000/api/pipeline?status=pending&firm_name=Test%20Firm"

Run research step:
curl -X POST http://localhost:3000/api/pipeline/{id}/research

Run legal resolution step:
curl -X POST http://localhost:3000/api/pipeline/{id}/legal-resolution

Run data extraction step:
curl -X POST http://localhost:3000/api/pipeline/{id}/data-extraction

Run complete pipeline (convenience route):
curl -X POST http://localhost:3000/api/pipeline/run-complete \
-H "Content-Type: application/json" \
-d '{"firm_name": "Test Firm"}'

Retry a failed step:
# Retry research
curl -X POST http://localhost:3000/api/pipeline/{id}/retry/research

# Retry legal resolution
curl -X POST http://localhost:3000/api/pipeline/{id}/retry/legal_resolution

# Retry data extraction
curl -X POST http://localhost:3000/api/pipeline/{id}/retry/data_extraction

Delete pipeline:
curl -X DELETE http://localhost:3000/api/pipeline/{id}
